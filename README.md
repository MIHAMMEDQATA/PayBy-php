# PayBy PHP SDK

## Introduce

payby.com

## Env

PHP >= 5.6

## Installation

### By Composer

```
composer require payby/payby-php
```

### Composer autoload import

```php
require_once('vendor/autoload.php');
```

### Manual import

```php
require_once('/path/to/payby-php/init.php');
```

## Api

### Initial

```php
\PayBy\PayBy::setPrivateKey('YOUR-KEY');
```
